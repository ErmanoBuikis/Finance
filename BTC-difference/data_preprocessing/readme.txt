

il file data_preprocessing.py comprende tutte le funzioni 
di eliminazione degli errori nei dati.
Comprende anche i vari metodi di normalizzazione dei dati.

Ci sono molteplici tipi di errore e molteplici
modi di normalizzare. Nella cartella dei documenti 
vengono trovate le spiegazioni di ciascun metodo con un file
.ipynb, che deve essere lanciato con jupyter-notebook o 
jupyter-lab per la sua visualizzazione.

viene preso sempre in input un file .csv dei dati
e viene restituito un altro file .csv con i dati processati
